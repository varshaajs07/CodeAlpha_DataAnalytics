# CodeAlpha Data Analytics Internship

## Task 2: Exploratory Data Analysis (EDA) & Task 3: Data Visualization

**Dataset used:** Titanic passenger dataset (loaded directly from the `seaborn` library — no download needed)

### Files
- `task2_eda.py` — explores the dataset structure, finds missing values, answers key questions (survival rate by class/gender/age), checks correlations, and detects outliers in fares.
- `task3_visualization.py` — creates 7 charts (bar charts, histogram, box plot, pie chart, and a mini dashboard) to visually tell the story of who survived the Titanic and why.
- `output_images/` — all charts generated by the two scripts.

### How to run
```bash
pip install pandas matplotlib seaborn
python task2_eda.py
python task3_visualization.py
```

### Key Insights
- Overall survival rate was about 38%.
- Women had a much higher survival rate (~74%) than men (~19%).
- 1st class passengers survived at a much higher rate (~63%) than 3rd class (~24%).
- `age` and `deck` columns had significant missing data.
- Fare and passenger class are strongly (negatively) correlated.
